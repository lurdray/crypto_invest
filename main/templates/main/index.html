How does OnpointCurrency Work?

Onpoint currency works in four simple steps for better user intaraction.

Validation: this comprises the sign up, on-time payment and KYC.

Commit: this stage immidiately follows after the the user has been verified and validated. in the state the user selects a package of his choice and after approval by the admin this user will be peer to another user to make his commit to, with a deadline for payment.

Withdrawal: after this user has successful made his commit, it's his turn to earn, in this stage a user will be peered to make commit to him also with a deadline.
